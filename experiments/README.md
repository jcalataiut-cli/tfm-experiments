# Experimentos TFM

Esta carpeta contiene los notebooks y scripts de experimentación numérica.

## Estructura

- `01_clt_convergence.ipynb` — Notebook sobre verificación CLT en redes neuronales
- `02_kernel_comparison.ipynb` — Notebook sobre comparación de kernels NNGP
- `compile_report.py` — Script para generar informe LaTeX unificado
- `fig*.png` — Figuras generadas por los experimentos
- `*_report.tex` — Fragmentos LaTeX para inclusión en informes
